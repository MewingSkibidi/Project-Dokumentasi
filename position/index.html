Semua Jenis Position di CSS

1. Position: Static (Default)

.static-box { position: static; }
Static adalah nilai default. Elemen mengikuti alur normal dokumen. Properti top, right, bottom, left tidak berpengaruh.
Static Box

2. Position: Relative

.relative-box { position: relative; top: 30px; left: 50px; }
Relative memposisikan elemen relatif terhadap posisi normalnya. Ruang asli elemen tetap dipertahankan.
Relative Box

3. Position: Absolute

.absolute-box { position: absolute; top: 50px; right: 30px; }
Absolute memposisikan elemen relatif terhadap ancestor terdekat yang positioned (bukan static). Elemen dikeluarkan dari normal flow.
Absolute Box

Ini adalah container dengan position: relative. Box absolute diposisikan relatif terhadap container ini.

4. Position: Fixed

.fixed-box { position: fixed; top: 10px; right: 10px; }
Fixed memposisikan elemen relatif terhadap viewport browser. Elemen akan tetap di posisi yang sama saat di-scroll.

Lihat kotak oranye di pojok kanan atas layar - itu adalah contoh position fixed!

5. Position: Sticky

.sticky-box { position: sticky; top: 10px; }
Sticky adalah gabungan relative dan fixed. Elemen berperilaku relative sampai mencapai threshold tertentu, lalu menjadi fixed.

Ringkasan Perbandingan

  • Static: Posisi normal, tidak dapat dipindah dengan top/left/etc
  • Relative: Dapat dipindah dari posisi normal, ruang asli tetap ada
  • Absolute: Diposisikan relatif terhadap parent yang positioned
  • Fixed: Diposisikan relatif terhadap viewport, tidak bergerak saat scroll
  • Sticky: Kombinasi relative dan fixed berdasarkan scroll position
