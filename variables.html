üéØ Pengenalan CSS Variables

CSS Variables (Custom Properties) adalah fitur CSS yang memungkinkan kita mendefinisikan nilai yang dapat digunakan kembali dan diubah secara dinamis dalam stylesheet.

Keuntungan CSS Variables:

  • Dinamis: Nilai bisa diubah runtime dengan JavaScript
  • Scoping: Bisa didefinisikan di elemen tertentu
  • Maintainable: Perubahan di satu tempat mempengaruhi seluruh stylesheet
  • Theming: Mudah membuat tema light/dark

üìù Basic Syntax

Deklarasi Variabel

/* Deklarasi di root element (global) */ :root { --primary-color: #667eea; --spacing: 1rem; --border-radius: 8px; } /* Deklarasi di element tertentu (scoped) */ .card { --card-bg: #ffffff; }

Menggunakan Variabel

.button { background-color: var(--primary-color); padding: var(--spacing); border-radius: var(--border-radius); } .card { background: var(--card-bg, #f8f9fa); /* Fallback value */ }

üí° Tips & Best Practices

1. Organisasi Variabel

/* Kelompokkan variabel yang terkait */ :root { /* Colors */ --primary: #667eea; --secondary: #764ba2; /* Spacing */ --space-sm: 0.5rem; --space-md: 1rem; /* Typography */ --text-base: 1rem; --text-lg: 1.25rem; }

2. Theming dengan Variabel

/* Light theme */ [data-theme="light"] { --bg: #ffffff; --text: #333333; } /* Dark theme */ [data-theme="dark"] { --bg: #121212; --text: #f8f9fa; } body { background: var(--bg); color: var(--text); }

3. Fallback Values

.element { color: var(--undefined-var, #ff0000); /* Fallback ke merah */ padding: var(--space-xl, var(--space-lg, 1rem)); /* Nested fallback */ }

üöÄ Kesimpulan

CSS Variables adalah alat yang powerful untuk membuat stylesheet yang lebih dinamis, maintainable, dan fleksibel.
