Apa itu CSS?

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan dan tata letak elemen-elemen pada halaman web. CSS memisahkan konten (HTML) dari presentasi (tampilan), sehingga desain website menjadi lebih rapi, konsisten, dan mudah diatur.

CSS memungkinkan kamu mengubah warna, ukuran, font, jarak, posisi, hingga membuat animasi pada elemen web. Dengan CSS, satu file style bisa digunakan untuk banyak halaman sekaligus, sehingga pengelolaan desain website menjadi lebih efisien.

Fungsi utama CSS:

  • Mengatur warna, font, dan ukuran teks.
  • Mengatur jarak (margin, padding) dan posisi elemen.
  • Membuat layout responsif yang menyesuaikan berbagai ukuran layar.
  • Menambahkan efek visual seperti hover, transisi, dan animasi.
  • Membuat tampilan website lebih menarik dan profesional.

CSS biasanya digunakan bersama HTML dan JavaScript untuk membangun website modern.


Contoh Sederhana CSS


<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Contoh CSS</title>
    <style>
    body {
        background-color: #f4f4f9;
        font-family: Arial, sans-serif;
        margin: 20px;
    }
    h1 {
        color: #2c3e50;
        text-align: center;
    }
    p {
        font-size: 18px;
        line-height: 1.6;
        color: #555;
    }
    </style>
</head>
<body>
    <h1>Belajar CSS</h1>
    <p>CSS membantu kita membuat halaman web menjadi lebih indah dan rapi.</p>
</body>
</html>
