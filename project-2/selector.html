üöÄ Pengenalan CSS Selectors

CSS Selectors adalah pola yang digunakan untuk memilih elemen HTML yang akan diberi styling. Selector adalah fondasi dari CSS yang memungkinkan kita mengontrol tampilan elemen-elemen spesifik di halaman web.

Video tutorial CSS Selectors untuk pemula

Dengan CSS Selectors, kita dapat menargetkan elemen berdasarkan nama tag, class, ID, atribut, posisi dalam dokumen, dan banyak kriteria lainnya.

üéØ Keunggulan CSS Selectors:

  • Presisi: Memungkinkan penargetan elemen yang sangat spesifik
  • Fleksibilitas: Berbagai cara untuk memilih elemen
  • Efisiensi: Satu selector bisa menargetkan banyak elemen
  • Maintainability: Mudah diubah dan dikelola
  • Reusability: Selector dapat digunakan kembali

üìê Selector Dasar

Mari kita pelajari selector-selector dasar yang paling sering digunakan:

1. Universal Selector (*)

Memilih semua elemen di halaman:

* { margin: 0; padding: 0; box-sizing: border-box; }

2. Element/Type Selector

Memilih semua elemen dengan tag tertentu:

p { color: blue; font-size: 16px; } h1 { color: red; font-size: 32px; } div { background-color: lightgray; padding: 20px; }

3. Class Selector (.)

Memilih elemen berdasarkan atribut class:

.highlight { background-color: yellow; font-weight: bold; } .container { max-width: 1200px; margin: 0 auto; } .btn { padding: 10px 20px; border: none; border-radius: 5px; cursor: pointer; }
<p class="highlight">Teks yang di-highlight</p> <div class="container">Container dengan max-width</div> <button class="btn">Tombol dengan styling</button>

4. ID Selector (#)

Memilih elemen berdasarkan atribut ID (harus unik):

#header { background-color: navy; color: white; padding: 20px; } #main-content { margin: 20px 0; } #footer { background-color: gray; text-align: center; }

5. Attribute Selector

Memilih elemen berdasarkan atribut yang dimiliki:

/* Elemen dengan atribut tertentu */ [type] { border: 2px solid blue; } /* Elemen dengan nilai atribut spesifik */ [type="text"] { background-color: white; } /* Elemen dengan nilai atribut yang mengandung kata tertentu */ [class*="btn"] { cursor: pointer; } /* Elemen dengan nilai atribut yang diawali kata tertentu */ [class^="nav"] { display: flex; } /* Elemen dengan nilai atribut yang diakhiri kata tertentu */ [class$="active"] { font-weight: bold; }

üîó Selector Kombinasi

Selector kombinasi memungkinkan kita memilih elemen berdasarkan hubungan mereka dengan elemen lain:

1. Descendant Selector (Spasi)

Memilih semua elemen yang merupakan keturunan dari elemen tertentu:

/* Semua p di dalam div */ div p { color: blue; } /* Semua a di dalam nav */ nav a { text-decoration: none; color: white; } /* Semua li di dalam ul yang ada di dalam nav */ nav ul li { list-style: none; display: inline-block; }

2. Child Selector (>)

Memilih elemen yang merupakan anak langsung dari elemen tertentu:

/* Hanya p yang merupakan anak langsung div */ div > p { margin-bottom: 10px; } /* Hanya li yang merupakan anak langsung ul */ ul > li { padding: 5px 0; } /* Hanya img yang merupakan anak langsung article */ article > img { width: 100%; height: auto; }

3. Adjacent Sibling Selector (+)

Memilih elemen yang langsung mengikuti elemen tertentu:

/* p yang langsung setelah h2 */ h2 + p { font-weight: bold; margin-top: 0; } /* img yang langsung setelah h1 */ h1 + img { margin-top: 20px; } /* div yang langsung setelah form */ form + div { padding-top: 10px; }

4. General Sibling Selector (~)

Memilih semua elemen sibling yang mengikuti elemen tertentu:

/* Semua p yang mengikuti h2 di level yang sama */ h2 ~ p { color: gray; } /* Semua div yang mengikuti header */ header ~ div { margin-top: 20px; } /* Semua input yang mengikuti label */ label ~ input { margin-left: 10px; }

5. Multiple Selector (Comma)

Memilih beberapa elemen sekaligus dengan aturan yang sama:

/* Styling yang sama untuk h1, h2, h3 */ h1, h2, h3 { font-family: 'Arial', sans-serif; margin-bottom: 15px; } /* Styling untuk semua elemen input */ input, textarea, select { border: 1px solid #ccc; padding: 8px; border-radius: 4px; } /* Styling untuk class tertentu */ .btn, .button, .submit { background-color: blue; color: white; padding: 10px 15px; }

üé≠ Pseudo-class Selectors

Pseudo-class selector memungkinkan kita memilih elemen berdasarkan state atau posisi mereka:

1. Interactive Pseudo-classes

/* Hover effect */ a:hover { color: red; text-decoration: underline; } /* Focus state untuk form elements */ input:focus { border-color: blue; outline: none; box-shadow: 0 0 5px rgba(0,0,255,0.3); } /* Active state */ button:active { transform: scale(0.98); } /* Visited links */ a:visited { color: purple; }
üñ±Ô∏è Interactive Demo

2. Structural Pseudo-classes

/* First child */ li:first-child { font-weight: bold; } /* Last child */ li:last-child { border-bottom: none; } /* Nth child - genap */ tr:nth-child(even) { background-color: #f2f2f2; } /* Nth child - ganjil */ tr:nth-child(odd) { background-color: white; } /* Nth child - pattern */ li:nth-child(3n) { color: red; /* setiap kelipatan 3 */ } /* Only child */ p:only-child { text-align: center; } /* Empty elements */ div:empty { display: none; }

3. Form Pseudo-classes

/* Input yang diperlukan */ input:required { border-left: 3px solid red; } /* Input yang valid */ input:valid { border-left: 3px solid green; } /* Input yang tidak valid */ input:invalid { border-left: 3px solid red; } /* Checkbox yang dicentang */ input:checked { transform: scale(1.1); } /* Input yang disabled */ input:disabled { opacity: 0.5; cursor: not-allowed; }

4. Negation Pseudo-class

/* Semua p kecuali yang memiliki class special */ p:not(.special) { color: gray; } /* Semua input kecuali yang bertipe submit */ input:not([type="submit"]) { margin-bottom: 10px; } /* Semua div kecuali yang pertama */ div:not(:first-child) { margin-top: 20px; }

üé® Pseudo-element Selectors

Pseudo-element memungkinkan kita menambahkan styling pada bagian tertentu dari elemen:

1. ::before dan ::after

/* Menambahkan konten sebelum elemen */ h2::before { content: "üìã "; color: blue; } /* Menambahkan konten setelah elemen */ .external-link::after { content: " üîó"; font-size: 0.8em; } /* Membuat decorative elements */ .quote::before { content: """; font-size: 2em; color: #ccc; position: absolute; left: -20px; top: -10px; } .quote::after { content: """; font-size: 2em; color: #ccc; }

2. ::first-letter dan ::first-line

/* Styling huruf pertama */ p::first-letter { font-size: 3em; float: left; line-height: 0.8; margin-right: 5px; color: #ff6b6b; } /* Styling baris pertama */ p::first-line { font-weight: bold; color: navy; }

3. ::selection

/* Styling teks yang diseleksi */ ::selection { background-color: #667eea; color: white; } /* Styling selection untuk Firefox */ ::-moz-selection { background-color: #667eea; color: white; }

4. ::placeholder

/* Styling placeholder text */ input::placeholder { color: #999; font-style: italic; opacity: 0.7; } textarea::placeholder { color: #666; font-family: serif; }

‚öñÔ∏è Specificity dan Priority

Specificity menentukan aturan CSS mana yang akan diterapkan ketika ada konflik. Semakin tinggi specificity, semakin prioritas aturan tersebut.

üèÜ Urutan Specificity (dari tertinggi ke terendah):

  1. Inline styles - style="..." (1000 poin)
  2. IDs - #myId (100 poin)
  3. Classes, attributes, pseudo-classes - .myClass, [type="text"], :hover (10 poin)
  4. Elements dan pseudo-elements - div, p, ::before (1 poin)

‚ö° !important Declaration

Deklarasi !important akan mengoverride semua aturan lain, terlepas dari specificity:

/* Aturan normal */ p { color: blue; } /* Aturan dengan specificity lebih tinggi */ #content p { color: red; } /* !important akan mengoverride semuanya */ p { color: green !important; } /* Hanya !important lain yang bisa mengoverride */ #content p { color: yellow !important; }

üí° Tips Mengelola Specificity:

  • Hindari penggunaan !important kecuali benar-benar diperlukan
  • Gunakan class selector daripada ID selector untuk styling
  • Buat selector sesederhana mungkin
  • Gunakan metodologi seperti BEM untuk menghindari konflik
  • Atur CSS dari general ke specific

üíé Tips dan Best Practices

1. Organisasi CSS yang Baik

/* 1. Reset/Normalize */ * { margin: 0; padding: 0; box-sizing: border-box; } /* 2. Base/Typography */ body { font-family: 'Arial', sans-serif; line-height: 1.6; color: #333; } /* 3. Layout */ .container { max-width: 1200px; margin: 0 auto; padding: 0 20px; } /* 4. Components */ .btn { display: inline-block; padding: 10px 20px; border: none; border-radius: 5px; cursor: pointer; text-decoration: none; } /* 5. Utilities */ .text-center { text-align: center; } .hidden { display: none; } .clearfix::after { content: ""; display: table; clear: both; }

2. Metodologi BEM (Block Element Modifier)

/* Block */ .card { background: white; border-radius: 8px; padding: 20px; } /* Element */ .card__title { font-size: 1.5em; margin-bottom: 10px; color: #333; } .card__content { color: #666; line-height: 1.6; } .card__button { background: blue; color: white; padding: 10px 15px; border: none; border-radius: 4px; } /* Modifier */ .card--featured { border: 2px solid gold; background: #fffbf0; } .card__button--large { padding: 15px 25px; font-size: 1.1em; }
<div class="card card--featured"> <h2 class="card__title">Featured Article</h2> <p class="card__content">This is a featured article with special styling.</p> <button class="card__button card__button--large">Read More</button> </div>

3. Performance Tips

  • Hindari selector yang terlalu kompleks:
/* Buruk - terlalu spesifik dan lambat */ body div.container section.content article.post p.paragraph span.highlight { color: red; } /* Baik - sederhana dan cepat */ .highlight { color: red; }
  • Gunakan ID untuk JavaScript, bukan styling:
/* Untuk JavaScript */ #modal-trigger { /* hanya untuk JS */ } /* Untuk styling */ .modal-button { background: blue; color: white; }

4. Responsive Design dengan Selectors

/* Mobile First Approach */ .nav-menu { display: none; } .nav-toggle { display: block; } /* Tablet dan Desktop */ @media (min-width: 768px) { .nav-menu { display: flex; } .nav-toggle { display: none; } .nav-menu li { margin-right: 20px; } .nav-menu li:last-child { margin-right: 0; } }

5. Accessibility dengan Selectors

/* Focus indicators yang jelas */ a:focus, button:focus, input:focus { outline: 2px solid #4A90E2; outline-offset: 2px; } /* Skip links */ .skip-link { position: absolute; top: -40px; left: 6px; background: #000; color: white; padding: 8px; text-decoration: none; transition: top 0.3s; } .skip-link:focus { top: 6px; } /* Responsive text */ @media (prefers-reduced-motion: reduce) { * { animation-duration: 0.01ms !important; animation-iteration-count: 1 !important; transition-duration: 0.01ms !important; } }

6. Debugging CSS Selectors

Tools dan teknik untuk debugging:

  • Gunakan browser DevTools untuk inspect elements
  • Tambahkan border sementara untuk melihat element boundaries
  • Gunakan outline untuk debugging tanpa mempengaruhi layout
  • Manfaatkan CSS validator untuk menangkap syntax errors
/* Debugging helper */ * { outline: 1px solid red !important; } /* Atau lebih spesifik */ .debug * { background: rgba(255, 0, 0, 0.1) !important; border: 1px solid red !important; }

Advanced CSS Selectors dan tips praktis

üìö Rangkuman Best Practices:

  • Gunakan class untuk styling, ID untuk JavaScript
  • Buat selector sesederhana mungkin
  • Hindari !important kecuali benar-benar diperlukan
  • Organisir CSS dari general ke specific
  • Gunakan metodologi seperti BEM untuk maintainability
  • Pertimbangkan performance dengan selector yang efisien
  • Test di berbagai browser dan device
