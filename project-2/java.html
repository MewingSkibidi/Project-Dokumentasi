Apa itu JavaScript?

JavaScript adalah bahasa pemrograman tingkat tinggi yang digunakan terutama untuk membuat halaman web menjadi interaktif dan dinamis. JavaScript berjalan di sisi klien (browser), sehingga dapat merespons aksi pengguna secara langsung tanpa perlu memuat ulang halaman.

JavaScript dapat digunakan untuk berbagai hal, seperti memanipulasi elemen HTML dan CSS, membuat animasi, memvalidasi form, mengambil data dari server tanpa reload (AJAX), hingga membangun aplikasi web modern (seperti SPA dengan React, Vue, atau Angular).

Fungsi utama JavaScript:

  • Membuat halaman web interaktif (misal: tombol, menu dropdown, slider, popup).
  • Memanipulasi dan mengubah isi dokumen HTML secara dinamis (DOM manipulation).
  • Memvalidasi input pengguna sebelum dikirim ke server.
  • Mengambil dan mengirim data ke server secara asynchronous (AJAX/fetch API).
  • Membuat animasi dan efek visual di halaman web.
  • Membangun aplikasi web modern (frontend dan backend dengan Node.js).

JavaScript biasanya digunakan bersama HTML dan CSS untuk membangun website yang modern dan responsif.


Contoh Sederhana JavaScript


<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Contoh JavaScript Sederhana</title>
    <style>
        body { font-family: Arial, sans-serif; margin: 20px; }
        #hasil { color: green; font-weight: bold; }
    </style>
</head>
<body>
    <h1>Belajar JavaScript</h1>
    <p>Klik tombol di bawah ini untuk menampilkan pesan:</p>
    <button onclick="tampilkanPesan()">Klik Saya!</button>
    <p id="hasil"></p>

    <script>
    // Fungsi untuk menampilkan pesan di halaman
    function tampilkanPesan() {
        document.getElementById('hasil').textContent = "Halo, kamu baru saja menekan tombol!";
    }
    </script>
</body>
</html>
