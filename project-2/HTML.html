Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa markup standar untuk membuat dan menyusun halaman web. HTML digunakan untuk menentukan struktur dasar dari sebuah website, seperti judul, paragraf, gambar, tautan, tabel, form, dan elemen-elemen lainnya.

HTML bukan bahasa pemrograman, melainkan bahasa markup yang mendeskripsikan struktur dan isi dari dokumen web. Setiap elemen HTML ditulis menggunakan tag, misalnya <p> untuk paragraf, <h1> untuk judul, dan <a> untuk tautan.

Fungsi utama HTML:

  • Menyusun struktur halaman web (header, footer, main content, sidebar, dsb).
  • Menampilkan teks, gambar, video, audio, dan berbagai media lainnya.
  • Membuat tautan (link) antar halaman atau ke website lain.
  • Membuat form untuk input data pengguna.
  • Menyusun tabel, daftar, dan elemen-elemen lain secara terstruktur.

HTML biasanya digunakan bersama CSS (untuk tampilan) dan JavaScript (untuk interaktivitas).


Contoh Sederhana HTML


<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Contoh HTML Sederhana</title>
</head>
<body>
    <h1>Belajar HTML</h1>
    <p>HTML adalah fondasi utama dalam membangun halaman web.</p>
    <ul>
        <li>HTML digunakan untuk membuat struktur web.</li>
        <li>Setiap elemen ditulis menggunakan tag, misalnya &lt;p&gt; untuk paragraf.</li>
        <li>HTML mudah dipelajari dan sangat penting untuk web developer.</li>
    </ul>
    <a href="https://www.w3schools.com/html/" target="_blank">Pelajari lebih lanjut tentang HTML</a>
</body>
</html>
